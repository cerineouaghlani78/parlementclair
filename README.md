# parlementclair
ParlementClair rend l’action parlementaire accessible grâce à l’IA et la data. Notre outil découpe les longs PDF des rapports officiels, utilise un LLM pour extraire une synthèse structurée (pour/contre, impacts) et convertit les chiffres en graphiques clairs, le tout centralisé sur une interface web intuitive pour les citoyens.
